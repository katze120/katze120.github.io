RHEL 7 new build

This page records how to build a RHEL7 machine, mostly manually for now, with TODO steps for future automation.

Did this before, but want to do it again and document the steps.

Steps:

  • Build RHEL7:

Create new VM and boot from CDROM rhel-server-7.1-x86_64-dvd.iso

TODO: create kickstart template or VM template

  • Bring up network:

    vi /etc/sysconfig/network-scripts/ifcfg-enp0s3
update ONBOOT=yes
systemctl restart network

  • Set hostname:

    hostnamectl set-hostname katzerhel7a

TODO: setup in-house DNS & DHCP server

  • [Optional] Set up ssh key from Ansible Controller Machine. From this point onwards, subsequent steps can be done via Ansible. More details in “ansible” blog post.

  • Create local repo:

Make sure boot order is now HDD before CDROM
Add rhel-server-7.1-x86_64-dvd.iso to CDROM

mkdir /media/dvd

vi /etc/fstab
insert below line 
/dev/sr0        /media/dvd      iso9660 loop    0 0

mount -a

cp -p /media/dvd/media.repo /etc/yum.repos.d/rhel7dvd.repo
vi /etc/yum.repos.d/rhel7dvd.repo
Update below config: 
gpgcheck=0
cost=500
enabled=1
baseurl=file:///media/dvd/
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-redhat-release
  • [Optional] Create EPEL repo:
rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm
